# ALPA Assets

Dieses Verzeichnis enthält Assets für ALPA-Projekte.

## Verzeichnisstruktur

```
/alpa/
  ├── assets/                # Assets
  │   ├── fonts/             # Schriftarten
  │   ├── icons/             # Icons
  │   └── logos/             # Logos
  │
  └── README.md              # Diese Datei
```

- Halten Sie sich an die Farbpalette und Gestaltungsrichtlinien von ALPA.one
